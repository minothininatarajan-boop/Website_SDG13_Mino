Climate Action Hackathon 2025 - Landing Page


## Project Overview
The Climate Action Hackathon 2025 landing page is designed to raise awareness about urban heat and flood resilience, aligning with SDG 13 (Climate Action). The website provides details about the event, the schedule, key speakers, and challenges. It serves as an informational hub for participants to stay updated.

## Features
- Hero Section: The page features an impactful Hero Section, introducing the event with a call-to-action button for registration.
- About the Event: A section explaining the mission and goals of the hackathon.
- SDG 13 Connection: Highlights how the event aligns with SDG 13: Climate Action.
- Event Schedule: A detailed schedule of the event with key dates.
- Speakers & Mentors: List of featured speakers and mentors.

- Resources: Provides useful resources and links for participants.
- Registration Form: A form for participants to register for the event.
- Sponsors & Partners: Logos of the event's sponsors and partners.

## Technology Used
- HTML5: For creating the structure and content of the landing page.
- CSS: For styling and layout design, using Flexbox for a responsive, mobile-first design.
